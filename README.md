# Multi-Core Neural Network

A custom neural network implementation in C++, designed to leverage multi-core processing for enhanced parallelism and performance.

# Key Features

Layer-Level Parallelism: Each layer runs as a separate process.

Node-Level Multithreading: Each neuron within a layer is implemented as a thread.

Efficient Execution: Designed for multi-core CPUs to maximize computation speed and concurrency.

Low-Level Control: Gives deeper insight into how neural networks function internally at the process/thread level.

# Technologies

Language: C++

Concurrency: Multi-threading & Multi-processing
